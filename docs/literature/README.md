# Literature Review

This directory contains literature reviews, paper summaries, and references for the quantum machine learning research project.

## Organization

- Create markdown files for each paper reviewed
- Use the format: `YYYY-MM-DD_author_title.md`
- Include summary, key findings, and relevance to project

## Suggested Papers

### Foundational
- Schuld, M., & Petruccione, F. (2018). Supervised learning with quantum computers
- Biamonte, J., et al. (2017). Quantum machine learning. Nature, 549(7671), 195-202

### Quantum Kernels
- Havlíček, V., et al. (2019). Supervised learning with quantum-enhanced feature spaces

### Variational Algorithms
- Cerezo, M., et al. (2021). Variational quantum algorithms. Nature Reviews Physics, 3(9), 625-644

### Quantum Neural Networks
- Farhi, E., & Neven, H. (2018). Classification with quantum neural networks on near term processors
